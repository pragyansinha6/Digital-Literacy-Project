# 📚 Digital Literacy Project README

## 🎯 **Project Overview**  
🚀 This comprehensive project fulfills my role as a **Digital Ambassador** (Reg. No: 25BCE10256) at **VIT Bhopal University**. As a 1st-year B.Tech Computer Science Engineering student (Batch 2025-2029), I focused on promoting **digital literacy and awareness** through course **CSE0001**. The project covers essential skills for navigating digital tools safely, professional platform usage, coding practice, communication etiquette, and cybercrime prevention—specifically targeting Indian college students vulnerable to UPI phishing scams.

## 📋 **Detailed Tasks Breakdown**

- **Safe Practices**: Strong/unique passwords + MFA 🔒, HTTPS verification (padlock icon) ✅, phishing avoidance.

Created checklists (5 Do's/Don'ts each):

### **Task 5: Cybercrime Awareness** ⚠️  
- **Phishing Case Study**: Fake bank email → credential theft → financial/emotional loss.  
- **"Stay Safe Online" Checklist** 🛡️ (8 steps for UPI): Never share PIN/OTP, verify URLs/links, enable 2FA, check statements, report to 1930 helpline.
